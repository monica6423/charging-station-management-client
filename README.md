# Development Instructions
- [Getting started](#getting-started)

## Getting started

To initialize the project, run:

```
nvm use 10.16.0
npm install
```

To start local developement, run:

```
npm run start
```

To build for production, run:

```
npm run build
```

There is no env variable or other setup needed.
